# 🛒 Lista de Compras em Python

Este é um projeto simples de linha de comando desenvolvido em Python que simula uma lista de compras. O usuário pode selecionar produtos, visualizar os preços e obter o valor total da compra.

---

## 📌 Objetivo

Este projeto foi criado para praticar conceitos fundamentais de programação em Python, como:

* Estruturas de dados (listas e dicionários)
* Laços de repetição (`while`, `for`)
* Condicionais (`if/else`)
* Entrada de dados com `input()`
* Formatação de saída

---

## 🚀 Funcionalidades

* Exibe uma lista de produtos disponíveis com preços
* Permite adicionar produtos à lista de compras
* Valida se o produto existe
* Mostra os itens adicionados de forma organizada
* Calcula automaticamente o valor total da compra

---

## 🧠 Tecnologias utilizadas

* Python 3

---

## ▶️ Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/raffaeccard/shopping-list-python.git
```

2. Acesse a pasta do projeto:

```bash
cd shopping-list-python
```

3. Execute o arquivo:

```bash
python main.py
```
